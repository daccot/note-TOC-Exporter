# note-TOC-Exporter
noteの記事ページ・編集ページから目次（TOC）を生成してクリップボードにコピーします

Privacy Policy

note TOC Exporter does not collect, store, or transmit any personal data.

The extension only reads heading information from note.com pages locally in the user's browser in order to generate a table of contents (TOC).

All processing is performed locally within the browser.

No data is transmitted to external servers and no user information is stored.

The extension may copy generated text to the user's clipboard when the user activates the extension.
