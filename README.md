# note-TOC-Exporter
noteの記事ページ・編集ページから目次（TOC）を生成してクリップボードにコピーします

note TOC Exporter Privacy Policy

This extension does not collect, store, or transmit any personal data.

The extension only reads heading information from note.com pages locally in the browser in order to generate a table of contents (TOC).

All processing occurs locally in the user's browser.  
No data is sent to external servers.

Generated text may be copied to the user's clipboard at the user's request.
