# Changelog

## v2.5.1 - 2026-04-26

- Improved Side Panel TOC navigation with H2-focused display.
- Added H2 child heading expand/collapse and bulk expand/collapse controls.
- Added fallback TOC generation from article headings when note's built-in TOC is unavailable.
- Improved filtering to exclude non-article headings.
- Added configurable wallpaper backgrounds.
- Improved copy completion guidance and business-standard Japanese wording.
- Updated build artifacts for Chrome Web Store submission.
- Added a Side Panel option to hide the `いいなと思ったら応援しよう！` support heading.
